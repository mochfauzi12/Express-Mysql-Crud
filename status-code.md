Berikut adalah daftar HTTP status code beserta kategori dan keterangan singkatnya:

1xx (Informational)
100 (Continue)
101 (Switching Protocols)

2xx (Successful)
200 (OK)
201 (Created)
204 (No Content)

3xx (Redirection)
301 (Moved Permanently)
302 (Found)
307 (Temporary Redirect)

4xx (Client Error)
400 (Bad Request)
401 (Unauthorized)
403 (Forbidden)
404 (Not Found)
409 (Conflict)

5xx (Server Error)
500 (Internal Server Error)
502 (Bad Gateway)
503 (Service Unavailable)
504 (Gateway Timeout)
